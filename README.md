# Micro C Compiler

This project is a lightweight implementation of a C compiler designed for educational purposes. It features:

- **Parsing**: Parses C code into an Abstract Syntax Tree (AST).
- **Semantic Checking**: Validates the semantic correctness of the AST.
- **Code Generation**: Translates the AST into target machine code.
- **Optimization**: Implements basic optimization techniques.

## Installation

To install the Micro C Compiler:

```bash
# Clone the repository
git clone https://github.com/discoveryos/mcc.git
cd mcc

# Build the compiler
make
```

## Usage

To compile a C file, use:

```bash
./mcc input.c -o output
```
